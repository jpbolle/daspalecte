Politique de Confidentialité

Dernière mise à jour : 11 janvier 2026

En résumé : Daspalecte respecte votre vie privée. Vos données de vocabulaire sont stockées uniquement sur votre ordinateur. Les textes que vous traduisez ou simplifiez sont traités de manière sécurisée et ne sont pas conservés.

1. Introduction

Bienvenue sur Daspalecte, une extension Chrome développée par Jean-Philippe Bolle (PedagokIT) pour aider les étudiants en Français Langue Étrangère (FLE), notamment les élèves du dispositif DASPA.

Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations personnelles lorsque vous utilisez notre extension.

2. Données collectées

2.1 Données stockées localement (sur votre ordinateur)

Les données suivantes sont stockées uniquement sur votre navigateur via Chrome Storage API et ne sont jamais envoyées à nos serveurs :

  • Liste de vocabulaire : Les mots que vous avez traduits et ajoutés à votre liste personnelle
  • Préférences utilisateur :
    • Langue de traduction sélectionnée
    • Langue maternelle pour la compréhension
    • État d'activation des outils (traducteur, compréhension)
  • Cache de traductions : Pour éviter les requêtes API redondantes

2.2 Données traitées temporairement

Lorsque vous utilisez les fonctionnalités de traduction ou de compréhension IA, les données suivantes sont envoyées à notre backend sécurisé :

  • Mots individuels pour traduction (ex : "maison")
  • Paragraphes de texte pour simplification (uniquement quand vous cliquez sur le bouton ✨)
  • Liste de mots sélectionnés pour génération d'exercices

Important : Ces données sont traitées en temps réel et ne sont pas conservées sur nos serveurs après traitement. Aucun historique n'est enregistré.

2.3 Données que nous ne collectons PAS

  • ❌ Votre identité (nom, email, adresse)
  • ❌ Votre historique de navigation
  • ❌ Les URLs des pages que vous visitez
  • ❌ Vos informations personnelles ou sensibles
  • ❌ Cookies de suivi ou analytics

3. Utilisation des données

3.1 Finalités

Nous utilisons vos données uniquement pour :

  1. Fournir les services de l'extension :
    • Traduire les mots dans votre langue maternelle
    • Simplifier les textes complexes avec l'IA
    • Générer des exercices pédagogiques personnalisés
  2. Améliorer votre expérience :
    • Sauvegarder vos préférences
    • Éviter les traductions en double (cache local)

3.2 Pas de revente ou partage

Nous ne vendons, ne louons et ne partageons jamais vos données avec des tiers à des fins commerciales ou publicitaires.

4. Services tiers et APIs

4.1 Backend sécurisé (Google Cloud Run)

Notre extension utilise un backend hébergé sur Google Cloud Run (région : Europe-West1) pour :

  • Traiter les appels API vers Google Translate
  • Traiter les appels API vers Claude (Anthropic)

Ce backend agit comme un intermédiaire sécurisé et ne stocke aucune donnée utilisateur.

4.2 APIs externes utilisées

  • Google Translate API : Pour les traductions mot à mot
    • Politique : Google Cloud Translation Data Usage
  • Claude API (Anthropic) : Pour les simplifications de texte et génération d'exercices
    • Politique : Anthropic Privacy Policy
    • Modèle utilisé : Claude 3 Haiku

Note : Ces services tiers ont leurs propres politiques de confidentialité. Les textes que vous envoyez pour traduction ou simplification peuvent être soumis à leurs conditions d'utilisation.

5. Sécurité des données

Nous prenons la sécurité de vos données très au sérieux :

  • Connexions HTTPS : Toutes les communications avec notre backend sont chiffrées
  • Google Secret Manager : Les clés API sont stockées de manière sécurisée
  • Pas de base de données : Aucune base de données permanente côté serveur
  • Stockage local uniquement : Vos listes de vocabulaire restent sur votre appareil

6. Vos droits

Vous avez le contrôle total sur vos données :

  • Accès : Vos données sont accessibles via le panneau de l'extension
  • Modification : Vous pouvez modifier ou supprimer des mots de votre liste à tout moment
  • Suppression complète :
    1. Allez dans chrome://extensions/
    2. Trouvez "Daspalecte"
    3. Cliquez sur "Supprimer"
    4. Toutes vos données locales seront effacées
  • Exportation : Contactez-nous pour obtenir une copie de vos données (vocabulaire uniquement, stocké localement)

7. Permissions de l'extension

Daspalecte demande les permissions Chrome suivantes :

  • storage : Pour sauvegarder votre vocabulaire et préférences localement
  • scripting : Pour injecter les fonctionnalités de traduction sur les pages web
  • activeTab : Pour accéder au contenu de la page active (uniquement quand vous utilisez l'extension)
  • host_permissions (<all_urls>) : Pour permettre l'utilisation sur tous les sites web que vous visitez

Ces permissions sont nécessaires pour que l'extension fonctionne correctement. Elles ne sont utilisées que pour les fonctionnalités décrites ci-dessus.

8. Utilisateurs mineurs

Daspalecte est conçu pour les élèves, y compris les mineurs. Nous ne collectons aucune information personnelle identifiable des utilisateurs, quel que soit leur âge.

Les enseignants et parents peuvent utiliser l'extension avec leurs élèves en toute sécurité.

9. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité occasionnellement. Toute modification sera :

  • Publiée sur cette page avec une nouvelle date de mise à jour
  • Communiquée via une mise à jour de l'extension si les changements sont significatifs

10. Cookies et suivi

Daspalecte n'utilise AUCUN cookie de suivi, analytics ou publicité.

Nous ne suivons pas votre comportement de navigation.
